# Step 5 - Core Feature Build-Out

Gallery grid
- Virtualized photo grid with incremental loading.
- Initial load shows the first 30 photos; scrolling loads additional pages of 30.
- Pull to refresh and error states.

Photo viewer
- Fullscreen viewer with swipe and pinch zoom.
- Share and download actions.

Albums
- Create, edit, and delete albums.
- Add or remove photos.

Search
- Query by date, location, and text.
- Recent searches and suggestions.

Backup and sync
- Auto upload toggle.
- Upload progress and retry UI.
- Status for last successful backup.

Sharing
- Create share link with expiry.
- Native share sheet integration.

Settings
- Upload quality, Wi-Fi only, cache size.
- Account and sign-out.

Deliverables
- Feature complete MVP screens.
- API wiring for all core actions.
